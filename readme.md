wordnet-dict is name for two scripts that parse the output of the WordNet database to provide basic, command line, versions of a dictionary and thesaurus. The script `dict` is for a dictionary and `thes` is for a thesaurus.

[WordNet](http://wordnet.princeton.edu/) is an electronic lexical database from Princeton University â€” it details relationships between words. Though it is not designed as a dictionary, it does provide elemental definitions plus contextual examples, of which a dictionary can be made. Word pronunciations are not available.
